Dynamic Traffic Management System

Existing systems cannot dynamically adjust traffic light timing in response to real time traffic, unexpected situations such as traffic accidents, natural calamities, or sudden incidents. With advances in technology, traffic data such as traffic volume, speed, and waiting time can now be gathered by cameras.

We aim to collect traffic data though image processing for which we aim to use Microsoft's Azure Computer Vision The image will be recieved already existing cameras installed on traffic lights. After collecting this data, we run our scheduling algorithm to change the duration of lights. The algorithm will take into account, the number of cars in a particular lane and dynamically change the traffic light duration. Also this algorithm with take into account emergency vehicles and let them pass first by allowing the traffic in that particular lane to go. This solves the main problem of time wastage and saves fuel, since there is efficient traffic control.
